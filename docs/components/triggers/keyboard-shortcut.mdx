# 快捷键触发

快捷键触发让你绑定一个键盘组合键，按下后在匹配页面一键启动工作流。

## 概述

主按键和修饰键分开配置——主按键选择字母、数字或功能键，修饰键勾选 Ctrl/Shift/Alt/Meta（Mac 的 Command）。配置面板会实时预览当前组合键效果。可通过 URL 匹配模式限定快捷键只在特定页面生效。

## 快速上手

1. 拖入「快捷键触发」节点到画布
2. 在「主按键」下拉中选择一个按键（如 `A`）
3. 勾选修饰键（如 `Ctrl` + `Shift`），界面会实时显示 `Ctrl + Shift + A`
4. 可选填写「URL 匹配模式」限定生效页面
5. 保存后按下组合键，流程立即执行

## 参数参考

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 主按键 | 下拉选择 | — | 必填。支持字母 A-Z、数字 0-9、F1-F12、方向键、Enter、Space、Tab、Escape、Backspace |
| 修饰键 | 多选 | — | 必填。多选：`Ctrl`、`Shift`、`Alt`、`Meta`（Windows 的 Win 键 / Mac 的 Command） |
| URL 匹配模式 | 文本 | — | Chrome match patterns 格式，不填则所有页面生效。参见 [URL 匹配模式](../../reference/match-patterns) |

## 常见问题

### 按下快捷键后没有反应

**现象**：配置了快捷键，但按下后流程不启动。

**原因**：快捷键与浏览器原生快捷键或其他扩展冲突，被优先拦截。

**解决**：更换一个更复杂的组合键（如三键组合 Ctrl+Shift+Alt+Key），或在 Chrome 扩展管理页面 `chrome://extensions/shortcuts` 中检查冲突。
