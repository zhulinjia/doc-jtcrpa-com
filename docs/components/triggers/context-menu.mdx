# 右键菜单

右键菜单触发器将工作流注册为浏览器右键菜单项，用户在页面上右键即可快速启动。

## 概述

安装并配置后，浏览器右键菜单会新增你设定的菜单项。点击后，对应工作流在当前页面自动执行。可通过 URL 匹配模式限定菜单项只在特定页面出现。

## 快速上手

1. 拖入「右键菜单」节点到画布
2. 填写「菜单项名称」——右键菜单中显示的文字
3. 可选填写「URL 匹配模式」限定在特定页面才显示该菜单项
4. 保存后在匹配的页面上右键即可看到菜单项

## 参数参考

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 菜单项名称 | 文本 | — | 右键菜单中显示的文字，必填 |
| URL 匹配模式 | 文本 | — | Chrome match patterns 格式，不填则所有页面生效。参见 [URL 匹配模式](../../reference/match-patterns) |

## 常见问题

### 右键菜单中看不到配置的选项

**现象**：配置了菜单项名称，但实际右键后找不到。

**原因**：流程未保存，或 URL 匹配模式限制了页面范围，当前页面 URL 不匹配。

**解决**：确认流程已保存；检查 URL 匹配模式是否与当前页面匹配；如果不确定，先留空让菜单项在所有页面出现。
