# 触发事件

触发事件节点手动向元素派发事件，用于解决页面事件未正常响应的问题。

## 概述

手动向元素派发事件，支持 9 种类型：单击、双击、悬浮、离开、按下按键、失去焦点、改变、获取焦点、输入。

选择"按下按键"时需配置按键，选择"改变"或"输入"时需填写文本内容。

## 快速上手

1. 拖入「触发事件」节点到画布
2. 填写「目标元素」CSS 选择器（必填）
3. 选择事件类型（必填），如"改变"
4. 如果选择了"按下按键"，还需配置按键；如果选择了"改变"或"输入"，还需填写文本内容
5. 执行
