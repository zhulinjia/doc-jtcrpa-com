# 提交表单

提交表单节点触发指定 `<form>` 元素的提交行为。

## 概述

触发指定 `<form>` 元素的提交。节点会自动查找表单内的提交按钮并点击；找不到按钮时直接调用原生提交。

支持打开新标签页提交并在新页面中继续执行。

<a href="/img/提交表单.png" target="_blank"><img src="/img/提交表单.png" alt="提交表单" style={{maxWidth: 480, width: '100%'}} /></a>

## 快速上手

1. 拖入「提交表单」节点到画布
2. 填写「标签名」（必填，需符合变量命名规范）
3. 填写「目标元素」CSS 选择器（必填），指向目标 `<form>` 元素
4. 如需在新标签页提交，开启对应开关
5. 执行

## 常见问题

### 提交后页面没有跳转或提示没有反应

节点执行成功但页面没有跳转或预期的成功提示。

**解决方法**：在提交前检查表单各字段是否已正确填写，确保必填项和校验规则都已满足。
