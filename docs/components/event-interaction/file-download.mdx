# 下载文件

下载文件节点从指定 URL 下载文件到本地。

## 概述

填写下载 URL 后，节点会自动发起下载。可选择性指定保存文件名（支持相对目录），不填则使用服务端返回的原始文件名。

<a href="/img/下载文件.png" target="_blank"><img src="/img/下载文件.png" alt="下载文件" style={{maxWidth: 480, width: '100%'}} /></a>

## 快速上手

1. 拖入「下载文件」节点到画布
2. 填写「下载 URL」（必填），如 `https://example.com/file.pdf`
3. 可选填写「保存文件名」：直接填文件名保存到默认目录（如 `report.pdf`）；填相对路径保存到指定子目录（如 `downloads/report.pdf`）
4. 执行

## 常见问题

### 下载的文件名不是预期的

下载完成后文件名不是自定义的名称。

**解决方法**：在「保存文件名」中填写期望的文件名（含扩展名），如 `report.pdf`。
