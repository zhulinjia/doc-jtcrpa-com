# 选择选项

选择选项节点操作 `<select>` 下拉框，按值或按显示文本选中选项。

## 概述

支持两种匹配模式：

- **按 Value 匹配** — 根据 `<option>` 的 `value` 属性值选中选项
- **按显示文本匹配** — 根据选项展示的文字选中选项

选择「匹配模式」后，在「选项值」中填写对应的值或文本即可。

## 快速上手

1. 拖入「选择选项」节点到画布
2. 填写「目标元素」CSS 选择器（必填），指向目标 `<select>` 元素
3. 选择匹配模式：按 Value 或按显示文本
4. 填写「选项值」——要选中的选项的值或文本
5. 执行

## 参数参考

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 目标元素 | 文本 | — | `<select>` 元素的 CSS 选择器，必填 |
| 匹配模式 | 下拉选择 | value | `value` — 按 option 的 value 属性匹配；`label` — 按 option 的显示文本匹配 |
| 选项值 | 文本 | — | 要选中的选项值或文本，支持 `{{变量名}}` 引用，详见[变量名](../../core-concepts/variables) |

## 常见问题

### 选项存在但提示"未找到匹配选项"

**现象**：下拉框中可以看到目标选项，但节点报错。

**原因**：匹配模式选错——用 `value` 模式去匹配显示文本，或反之。

**解决**：在开发者工具中检查目标 `<option>` 元素，确认 `value` 属性和显示文本分别是什么，选择对应的匹配模式。
