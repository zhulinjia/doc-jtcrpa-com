# 读剪切板

读取你刚才复制（Ctrl+C）的文字内容，存到变量里给后面的节点用。

## 概述

复制一段文字后，用这个节点把它取出来。需要搭配右键菜单、快捷键这类手动触发的流程使用，定时任务没有权限读取。

<a href="/img/读剪切板.png" target="_blank"><img src="/img/读剪切板.png" alt="读剪切板" style={{maxWidth: 480, width: '100%'}} /></a>

## 快速上手

1. 拖入「读剪切板」节点到画布
2. 填写「输出变量」名称（必填），如 `复制的文字`
3. 后面的节点用 `{{复制的文字}}` 就能拿到刚才复制的内容
4. 流程必须通过右键菜单或快捷键触发，定时执行会失败

## 常见问题

### 读取失败、提示没有权限

**解决方法**：检查流程是不是通过右键菜单或快捷键触发的，不要用定时触发。
