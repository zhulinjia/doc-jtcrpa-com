# 写剪切板

把文字写入剪贴板，之后可以用 Ctrl+V 粘贴到任何地方。

## 概述

把指定文字写入剪贴板，后面接「按下按键」节点按 Ctrl+V 就能粘贴到输入框里。

填写内容时支持 `{{变量名}}` 引用之前的变量，详见[变量名](../../core-concepts/variables)。

## 快速上手

1. 拖入「写剪切板」节点到画布
2. 在「写入内容」中填写要写入的文字（必填），可用 `{{变量名}}` 引用
3. 后面接「按下按键」节点，选择 Ctrl+V 粘贴

## 常见问题

### 写入后粘贴没有内容

**解决方法**：先打开记事本按 Ctrl+V，看能不能粘出来。能粘出来说明是目标页面的问题，用「按下按键」节点发 Ctrl+V 组合键来粘贴。
