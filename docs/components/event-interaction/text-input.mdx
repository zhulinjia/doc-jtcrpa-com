# 输入文字

输入文字节点向输入框或文本域填写内容，支持模拟逐字输入和输入后的键盘操作。

## 概述

向输入框或文本域填写内容。填写后可选执行换行操作：直接输入、输入后按回车（Enter）、输入后按 Shift+Enter、输入后按 Tab。

如果选择器指向 `<label>` 或包裹 `<div>`，节点会自动找到其中真正的 `<input>` 或 `<textarea>` 元素。

**模拟逐字输入**：开启后逐字符输入，关闭则一次性填入。

## 快速上手

1. 拖入「输入文字」节点到画布
2. 填写「目标元素」CSS 选择器（必填），指向 `<input>` 或 `<textarea>`
3. 填写要输入的文本内容
4. 选择换行方式：无 / 回车 / Shift+回车 / Tab
5. 如需模拟真人打字，开启「模拟逐字输入」

## 常见问题

### 输入框显示了文字但表单校验不通过

文字已填入输入框但提交时校验提示不消失，通常是前端框架未检测到值变化。

**解决方法**：开启「模拟逐字输入」；或在输入文字后追加「触发事件」节点，手动派发 `input` 事件。
