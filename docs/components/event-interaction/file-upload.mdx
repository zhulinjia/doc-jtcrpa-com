# 上传文件

上传文件节点将本地文件注入到 `<input type="file">` 元素中，实现自动化文件上传。

## 概述

将文件上传到 `<input type="file">` 元素，支持本地文件和远程文件（URL）。

**使用前需在扩展管理中开启文件访问权限**。

文件路径支持 `{{变量名}}` 引用，详见[变量名](../../core-concepts/variables)。

## 快速上手

1. 拖入「上传文件」节点到画布
2. 填写「文件输入框选择器」CSS 选择器（必填），指向目标 `<input type="file">`
3. 填写「文件路径」：本地文件填绝对路径（如 `C:\data\report.pdf`），远程文件填 URL（如 `https://example.com/file.pdf`）
4. 执行

## 常见问题

### 上传时报文件路径不可访问

1. 打开浏览器扩展管理页面，找到本插件，点击「详细信息」
2. 开启「允许访问文件 URL」选项
3. 确认填写的路径是本地绝对路径（如 `C:\Users\用户名\Documents\file.pdf`），不是相对路径，且文件确实存在
4. 如果使用远程文件，确认 URL 完整且可以直接在浏览器中访问，文件可正常下载
