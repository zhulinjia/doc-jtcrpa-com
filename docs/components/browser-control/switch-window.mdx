# 切换窗口

切换窗口节点用于将浏览器焦点切换到另一个已打开的窗口。

## 概述

在「目标窗口」下拉中选择要切换到的窗口，执行即可。

<img src="/img/切换窗口.png" alt="切换窗口节点参数配置面板" width="520" />

## 使用方法

拖入节点后，在「目标窗口」下拉中选择要切换到的窗口。下拉列表中的选项来源如下：

| 选项 | 来源 |
|------|------|
| **当前窗口** | 始终存在，指当前激活的窗口 |
| **初始窗口** | 始终存在，指流程启动时已打开的窗口 |
| **其他窗口名称** | 由流程中上游节点的「标签名」决定 |

### 窗口名称从哪里来

下拉列表中除了「当前窗口」和「初始窗口」之外，其余选项的名称来自上游「[打开窗口](./open-window)」节点中填写的**标签名**。也就是说，你想切换到哪个窗口，就取决于上游打开它时给它起的名字。

开启「**激活窗口**」可在切换后自动将窗口聚焦到前台。

## 参数参考

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 目标窗口 | 下拉选择 | — | 选择一个已打开的窗口进行切换，选项由系统动态生成 |
| 激活窗口 | 开关 | false | 切换到目标后是否自动聚焦 |

## 常见问题

### 切换窗口后元素找不到

**现象**：切换到指定窗口后，后续节点报"找不到元素"。

**原因**：目标窗口的页面 URL 与预期不符，或页面尚未加载完成。

**解决**：切换后插入「[等待元素](../general-components/wait-element)」或「[打开网页](./open-page)」节点，确保目标窗口处于预期状态。
