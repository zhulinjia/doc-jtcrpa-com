# 页面截图

页面截图节点用于截取当前页面的指定区域，将结果保存为图片。

## 概述

提供三种截图模式：可视区域（当前屏幕可见部分）、整个页面（自动滚动拼接长图）、元素区域（截图指定 DOM 元素）。

## 使用方法

拖入节点后，选择「截图区域」模式：

### 可视区域

截取当前浏览器视口内的可见内容。适合只需要当前屏幕画面的场景，速度最快。

### 整个页面

自动滚动页面并逐屏截取，最终拼接为一张完整长图。适合截取需要滚动才能看全的长页面。

> 内部有安全限制——最大截图高度 25000px、最多截取 30 屏。超过任一上限会自动停止，防止无限滚动页面导致流程卡死。

### 元素区域

截取指定 DOM 元素所在的矩形区域。选择此模式后需要填写「**目标元素**」——用 [CSS 选择器](/core-concepts/css-selectors) 定位要截图的元素。

## 参数参考

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 截图区域 | 下拉选择 | — | `可视区域` — 当前视口；`整个页面` — 滚动拼接长图；`元素区域` — 指定元素 |
| 目标元素 | CSS 选择器 | — | 截图区域为「元素区域」时必填 |
| 文件名 | 文本 | — | 保存的文件名，支持 `{{变量名}}` 引用 |
