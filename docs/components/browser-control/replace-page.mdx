# 替换页面

替换页面节点在当前标签页中静默替换 URL，不添加浏览器历史记录。

## 概述

与「[打开网页](./open-page)」不同，替换页面使用 `location.replace()` 导航，不会在浏览器历史中留下记录，用户无法通过前进/后退回到替换前的页面。

<img src="/img/替换页面.png" alt="替换页面节点参数配置面板" width="520" />

## 使用方法

拖入节点后，填写目标 URL 即可。与「打开网页」的关键区别：

- **无历史记录**：使用 `location.replace()`，替换后用户按后退键不会回到替换前的页面。适合登录后跳转、表单提交后重定向等不希望用户返回的场景。
- **在当前标签页中执行**：不会打开新窗口或新标签页。

开启「**激活标签**」可将浏览器焦点切换到该标签页。

## 参数参考

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 页面地址 | 文本 | — | 替换后的 URL，必填。支持 `{{变量名}}` 引用，详见[变量名](../../core-concepts/variables) |
| 激活标签 | 开关 | false | 是否自动切换到该标签页 |
