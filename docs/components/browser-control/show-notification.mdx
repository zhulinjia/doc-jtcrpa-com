# 显示通知

显示通知节点用于在浏览器中显示一条系统级通知。

## 概述

向用户发送标题和内容组成的通知消息，适合长时间自动化任务中的进度反馈或异常提醒。**使用前需确保浏览器通知权限已授予**。

<img src="/img/显示通知.png" alt="显示通知节点参数配置面板" width="520" />

## 使用方法

拖入节点后，填写通知内容即可：

- **通知标题**：必填，通知的主标题。支持 `{{变量名}}` 引用变量值。
- **通知内容**：通知正文。支持 `{{变量名}}` 引用。

执行后通知会在操作系统层面弹出，即使浏览器在后台也能看到。适合在长流程的关键节点（如采集完成、异常捕获）加上通知，让用户无需一直盯着屏幕。

:::warning
使用前需确保浏览器已授予通知权限，否则通知不会弹出。详见下方常见问题。
:::

## 参数参考

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 通知标题 | 文本 | — | 必填。支持 `{{变量名}}` 引用，详见[变量名](../../core-concepts/variables) |
| 通知内容 | 文本 | — | 通知正文。支持 `{{变量名}}` 引用 |

## 常见问题

### 通知没有弹出

**现象**：节点执行成功但屏幕上没有看到通知。

**原因**：浏览器通知权限未授予，或操作系统通知中心对浏览器设置了静音。

**解决**：在浏览器地址栏左侧点击锁定图标 → 通知 → 选择「允许」。
