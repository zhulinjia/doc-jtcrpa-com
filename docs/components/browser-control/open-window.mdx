# 打开窗口

打开窗口节点在浏览器中创建一个新窗口。

## 概述

支持四种窗口状态：普通、全屏、最大化、最小化。普通模式下可自定义窗口位置（上/左）和大小（宽/高）。可选择激活窗口和将后续操作切换到新窗口执行。

## 快速上手

1. 拖入「打开窗口」节点到画布
2. 填写「窗口名」（必填，需符合[变量命名规范](../../core-concepts/variables)），后续可被「切换窗口」「关闭窗口」等节点引用
3. 选择窗口状态（默认普通）
4. 普通模式下可选配置位置（上/左）和大小（宽/高）
5. 根据需要开启激活窗口或切换到新窗口

## 参数参考

### 通用参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 窗口名 | 文本 | — | 节点名称，必填。需符合[变量命名规范](../../core-concepts/variables) |
| 窗口状态 | 下拉选择 | normal | `normal` — 普通（可调位置大小）；`fullscreen` — 全屏；`maximized` — 最大化；`minimized` — 最小化 |
| 激活窗口 | 开关 | false | 是否自动聚焦到新窗口 |
| 页面将在新窗口打开 | 开关 | false | 后续节点是否在新窗口中执行 |

### 普通模式专属参数（窗口状态为 normal 时显示）

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 上 | 数字 | 0 | 窗口距屏幕上边缘的像素 |
| 左 | 数字 | 0 | 窗口距屏幕左边缘的像素 |
| 宽 | 数字 | — | 窗口宽度（像素），默认为屏幕宽度 |
| 高 | 数字 | — | 窗口高度（像素），默认为屏幕高度 |
