# 刷新页面

刷新页面节点用于重新加载当前标签页。

## 概述

刷新页面作用于当前激活的标签页，无需选择目标。开启「绕过缓存」可强制刷新（相当于 Ctrl+F5）。

<img src="/img/刷新页面.png" alt="刷新页面节点参数配置面板" width="520" />

## 使用方法

拖入节点后，根据需要开启「绕过缓存」：

- **绕过缓存**：开启后忽略浏览器缓存，强制从服务器重新请求页面。相当于手动按 Ctrl+F5。适用于需要拿到最新数据的场景。

## 参数参考

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 绕过缓存 | 开关 | false | 开启后忽略缓存强制重新请求（相当于 Ctrl+F5） |

## 常见问题

### 强制刷新后页面数据未变化

**现象**：绕过缓存刷新后数据与刷新前相同。

**原因**：服务端返回的数据本身没有变化，或 CDN 层缓存未同步。

**解决**：在开发者工具 Network 面板确认请求状态码为 200 而非 304。
