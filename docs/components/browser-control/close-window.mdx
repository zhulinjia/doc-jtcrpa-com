# 关闭窗口

关闭窗口节点关闭指定的浏览器窗口。

## 概述

下拉框动态列出可选目标：始终包含「当前窗口」和「初始窗口」，同时自动识别流程中上游「打开窗口」节点创建的窗口，将它们也加入可选列表。

## 快速上手

1. 拖入「关闭窗口」节点到画布
2. 在「关闭目标」下拉中选择要关闭的窗口
3. 执行

## 参数参考

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 关闭目标 | 下拉选择 | — | 下拉选项动态生成，包含当前窗口、初始窗口，以及上游「打开窗口」节点创建的窗口 |

## 常见问题

### 关闭窗口后流程中断

**现象**：关闭窗口后后续节点报错。

**原因**：后续节点需要在已关闭的窗口中操作。

**解决**：关闭前先使用「切换窗口」节点将上下文转移到其他窗口。
