# 打开网页

打开网页节点在浏览器中导航到指定 URL，是最常用的导航节点。

## 概述

导航到目标 URL 后可选择：激活标签（切换到该标签页）、切换到新页面执行（后续节点在新页面中运行）。

<img src="/img/打开网页.png" alt="打开网页节点参数配置面板" width="520" />

## 参数参考

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 标签名 | 文本 | — | 节点名称，必填。需符合[变量命名规范](../../core-concepts/variables)，后续可被「切换页面」「关闭网页」等节点引用 |
| 页面地址 | 文本 | — | 目标 URL，必填。支持 `{{变量名}}` 引用，详见[变量名](../../core-concepts/variables) |
| 激活标签 | 开关 | false | 是否自动切换到该标签页 |
| 切换到新页面执行 | 开关 | false | 后续节点是否在新打开的页面中执行 |
