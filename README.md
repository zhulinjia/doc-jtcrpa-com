# 九头虫 RPA (Jitouchong RPA)

九头虫 RPA 是一款专为高效数据采集、网页自动化与业务流程解耦设计的浏览器原生自动化插件。基于可视化画布与节点流式架构，用户无需编写复杂的自动化脚本，通过简单的拖拽组合即可快速构建具备跨 Frame 执行、低延迟响应及深度 DOM 监控能力的自动化工作流。

---

## 核心优势

- **可视化流式编辑器**：基于直观的节点图画布，完美支持拖拽式流程构建、条件分支分流及多路并行处理。
- **深度 DOM 监控与智能匹配**：内置强大的多模态元素查询引擎，支持动态节点监听、跨 `iframe` 深层穿透。
- **高性能架构**：底层采用纯内存日志缓冲与极致优化的状态管理，完美应对每秒 100+ 次的高频数据触发流，确保 UI 界面丝滑无卡顿。
- **强大的企业级专属节点**：除基础事件外，全面支持 WebSocket 长连接、SSE 流式指令、网络层请求拦截（XHR/Fetch）等高级特性，轻松绕过前端混淆，直达底层数据。

---

## 全景组件生态

系统内置 6 大核心模块，涵盖从触发、控制到交互的全生命周期执行引擎：

### 1. 触发器

用于初始化和引导工作流的启动 ：

- **主动触发**：支持 **手动执行** 、**右键菜单** 快速清洗、**快捷键触发** 等高频常驻操作。
- **被动/信号触发**：支持分布式 **信号触发** 、**WebSocket** 长连接下发指令以及 **SSE 触发** 流式控制。
- **高级智能监听**：包含 **元素监控**（DOM 节点的出现/更新/消失）、**网络拦截**（直接拦截 XHR/Fetch 并提取 JSON） 与 **URL 匹配** 页面跟随。
- **定时任务**：支持 **定时触发** 周期化启动任。

### 2. 通用组件

支撑流程内部数据管理、沙箱运算与数据持久化 ：

- **数据治理**：包含 **设置变量**（声明全局/会话级变量） 与 **数据采集**（支持 CSS，提取文本、属性、视频或图片）。
- **异步协同**：**等待信号** 管道放行 ，**等待元素** 动态渲染加载 。
- **控制台审计**：支持 **打印输出**。
- **沙箱扩展**：支持 **执行代码** （嵌入原生 JavaScript 异步块） 与 **读取文件**（本地文件/远程文件）。

### 3. 流程控制 (Flow Control)

管理自动化路线规划与熔断机制 ：

- **分支与聚合**：**汇聚节点** 承接多条先导分支 ；**条件分支** 与 **多路分支** 实现复杂的逻辑分流。
- **循环与嵌套**：**循环执行** 遍历表格或数组 ，**调用流程** 嵌套子级 RPA 文件实现模块解耦 。
- **执行干预**：包含 **延迟** 、**停止** 、**暂停** 、**继续** 、**跳过** 、**跳出** 与 **返回** 。
- **信号交互**：支持 **发送信号** 唤醒其他挂起流程 。

### 4. 浏览器控制 (Browser Control)

管理浏览器实例化容器与标签页生命周期 ：

- **沙箱隔离**：**打开窗口**（支持无痕模式沙箱隔离）与 **关闭窗口**。
- **导航交互**：**打开网页**、**关闭网页** 、**页面前进** 、**页面后退** 、**刷新页面** 及 **替换页面** 。
- **深层定位**：**切换窗口** 、**切换页面**（按索引或URL） 以及 **切换 iframe**。
- **系统层级交互**：包含 **显示通知** 与 **处理弹窗** 。

### 5. 事件交互

深度模拟物理手势与高保真信息输入 ：

- **物理模拟**：**点击事件**（左/右/中键与双击）、**悬浮事件**（Hover 触发二级菜单） 与 **按下按键**（发送回车、退格等）。
- **流式交互**：**滚动页面**（模拟物理鼠标平滑滚轮逻辑） 与 **选择选项**（级联器适配）。
- **精准灌入**：**输入文字**（支持配置模拟打字延迟防止风控）、**勾选选框** 、**提交表单** 与 **触发事件** 调用原生 JS 事件 。
- **高级外设拓展**：**上传文件** （解除原生安全限制）、**下载文件** （接管下载任务并暂存） 以及 **读剪切板** / **写剪切板** 。

### 6. 网络组件 (Network Components)

提供更底层的 Web 请求和 HTTP 状态响应流监控 ：

- **等待 HTTP** ：持续监听底层网络会话，捕获目标接口响应并自动提取结构化返回体 。
- **发送 HTTP** ：绕过复杂的网页 UI 交互逻辑，直接通过后台发出集成 Cookie 的标准 REST/GraphQL 请求 。
