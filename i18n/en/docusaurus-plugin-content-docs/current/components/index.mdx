## Recommendations by User Level

### Getting Started

Master these first — they cover 80% of daily tasks:

1. [Manual Execute](./triggers/manual-execute) — Understand how workflows run
2. [Open Page](./browser-control/open-page) — Navigate to target pages
3. [Click](./event-interaction/click) — Click buttons and links
4. [Text Input](./event-interaction/text-input) — Fill in forms
5. [Data Collection](./general-components/data-collection) — Extract page content
6. [Set Variable](./general-components/set-variable) — Store data for later use
7. [Print Output](./general-components/print-output) — Check variable values while debugging

### Need Batch Processing

Once data volume grows, add loops and conditionals:

8. [Loop](./flow-control/loop) — Iterate through lists one by one
9. [Condition Branch](./flow-control/condition-branch) — Take different paths by condition
10. [Delay](./flow-control/delay) — Control frequency to avoid rate limiting

### Advanced

For finer control and cross-page orchestration:

11. [Switch iframe](./browser-control/switch-iframe) — Operate on embedded pages
12. [Element Monitor](./triggers/element-monitor) — Wait for dynamic content to appear before processing
13. [Call Flow](./flow-control/call-flow) — Split into reusable sub-workflows
14. [Network Intercept](./triggers/network-intercept) — Get API data directly
15. [Scheduled Trigger](./triggers/scheduled-trigger) — Unattended auto-execution
